# 🛒 Punto de Venta (POS) — JavaFX

Aplicación de escritorio desarrollada en **JavaFX** para la gestión de ventas, inventario y operaciones comerciales en tiempo real.

---

## 🚀 Descripción

El **Sistema Punto de Venta (POS)** es una aplicación diseñada para facilitar el proceso de ventas en negocios, permitiendo registrar productos, procesar compras y llevar control del inventario de manera eficiente.

Cuenta con una interfaz gráfica moderna gracias a **JavaFX**, ofreciendo una experiencia intuitiva para el usuario.

---

## 🎯 Características principales

* 🧾 **Ventas**

  * Registro rápido de productos
  * Cálculo automático del total
  * Generación de tickets

* 📦 **Inventario**

  * Control de stock
  * Actualización automática al vender
  * Gestión de productos

* 👤 **Usuarios**

  * Inicio de sesión
  * Roles básicos (Administrador / Cajero)

* 💰 **Caja**

  * Apertura y cierre de caja
  * Control de ingresos

* 📊 **Reportes**

  * Historial de ventas
  * Reportes diarios

* 🖥️ **Interfaz gráfica**

  * Diseño moderno con JavaFX
  * Navegación intuitiva

---

## 🛠️ Tecnologías utilizadas

* ☕ Java (JDK 8+)
* 🎨 JavaFX
* 🗄️ Base de datos: MySQL / SQLite
